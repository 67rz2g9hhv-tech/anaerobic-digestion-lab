# 🧪 Anaerobic Digestion Lab

An interactive PhET-style educational simulation teaching students how anaerobic digestion works.

## Features

- **Stage 1: Introduction** - Learn the basics of anaerobic digestion with animated visualizations
- **Stage 2: Experimentation** - Manipulate operating conditions and observe real-time feedback
- **Stage 3: Analysis** - Compare trials and analyze trends in methane production and system health
- **Stage 4: Challenge Mode** - Complete realistic engineering scenarios

## Visit the Simulation

[Launch the Lab](https://67rz2g9hhv-tech.github.io/anaerobic-digestion-lab/)

## How to Use

1. Open `index.html` in any modern web browser
2. Follow the guided stages to learn about anaerobic digestion
3. Experiment with different parameters to see how they affect methane production
4. Complete challenge scenarios to test your understanding

## Technical Details

- Pure HTML5, CSS3, and JavaScript (no external dependencies)
- Responsive design for desktop and tablet
- Based on simplified but realistic anaerobic digestion principles
